health-finder
=============

Implements a basic Health Finder developer API 

This is a work in progress. 

A preview can be seen at: http://karthikr.altervista.org/health-finder/


Libraries:

jQuery 1.11.1
Bootstrap - 3.2.0
FontAwesome - 4.1.0
Mustache js
